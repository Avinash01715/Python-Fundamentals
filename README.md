# Python Fundamentals
 
Practice code covering core Python concepts, organized folder by folder.

## Purpose
 
Personal practice log, part of a self-directed AI/ML learning path.

## Contents
 
- `01_basics` — syntax, variables, data types
- `02_conditionals` — if/else logic
- `03_loops` — for and while loops
- `04_functions` — functions and arguments
- `05_oop` — classes and objects
- `06_decorators` — decorators
- `08_handling_api` — working with APIs
- `09_virtual_env.py` — virtual environments
- `10_jupyter_notebook` — Jupyter notebook practice

## 🚀 Projects
 
A YouTube video manager (CRUD app) built three times, each with a different storage backend:
 
| Project | Storage |
|---|---|
| `07_Projects/1. Youtube Video Manager` | Plain text file |
| `07_Projects/2. Youtube Video Manager with sqlite3 db` | SQLite database |
| `07_Projects/3. Youtube Video Manager with monogodb` | MongoDB |
 
Each version implements the same Create/Read/Update/Delete workflow for video records, showing the same problem solved with progressively more capable storage.

## Usage
 
```bash
git clone https://github.com/Avinash01715/Python-Fundamentals.git
cd Python-Fundamentals
```
 
Browse into any folder and run the scripts with Python 3.
